## Usage

```bash
$ ./run_<script>.sh [-t] [-p] [-d dimension] [-k num-of-clusters]
  -t  run script in test mode
  -p  run preprocessing
  -c  run char2vec 
  -d  set char2vec vector dimension
  -k  k-means: set k clusters; knn: set k nearest neighbors
  -a  clustering algorithm 
  -s  segmentation algorithm
```
